# Button—按钮

查看代码 
<template>
  <div>
    //普通按钮
    <div>
      <s-button>点击按钮</s-button>
    </div>
    //普通按钮
    <s-button icon="setting">点击按钮</s-button>
    <s-button icon="setting" icon-position="right">点击按钮</s-button>
    //点击按钮加载中
    <div>
      <s-button :loading="loading" @click="loading = !loading">点击按钮</s-button>
    </div>
    //按钮组合
    <s-button-group>
      <s-button icon="left">上一页</s-button>
      <s-button>更多</s-button>
      <p-button icon="right" icon-position="right">下一页</p-button>
    </s-button-group>
  </div>
</template>

TIP

通过设置Button的属性来产生不同的按钮样式,按钮的属性说明如下：

属性 说明 类型 默认值
icon 引入icon（仅支持本框架提供的icon） String 详见组件Icon
:loading 绑定loading值 Boolean false
icon-position 设置icon方向 String left
