# Input-输入框

查看示例代码 
<template>
    //普通输入
    <s-input value="张三"></s-input>
    //禁止输入
    <s-input value="李四" disabled></s-input>
    //输入提示
    <s-input value="王" error="姓名不得少于两个字"></s-input>
    //父组件数据传入
    <s-input v-model="message"></s-input>
    <p>父组件:{{ message }}</p>
    <s-button @click=" message += 'hi' ">点击传入</s-button>
</template>
属性 说明 类型 默认值
value 绑定值 String -
error 绑定info值 String -
v-model 输入框内容 String -
disabled 禁用 Boolean false
