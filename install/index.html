# 安装

ScrewUI 提供了 Button,Input,Collapse 等常用组件，并在持续更新中，

适合移动端和 PC 端使用。现在开始安装使用吧！

# NPM

npm install screwui-vue2
  • 如果你使用的是 Vue CLI 搭建的项目，默认使用的包管理工具是 yarn,可忽略此提示。 如果此前以使用 npm 安装所需依赖，请rm -rf node_modules 然后使用 yarn 重新安装依赖，以免出错。

# YARN

yarn add screwui-vue2
